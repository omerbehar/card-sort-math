# Design Artifacts

Design documents produced by the `.claude` framework skills (e.g. `/design-system`,
`/quick-design`, `/brainstorm`, `/map-systems`). Conventional subfolders:

- `gdd/` — full Game Design Documents per system
- `quick-specs/` — lightweight specs for small changes
- `systems-index.md` — system decomposition & dependency map

See `docs/GAME_PLAN.md` for the overall product plan.
